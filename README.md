## サービス概要

OshiPitchは、Vtuberの名前を入力するだけで「布教用プレゼン資料」を自動生成するWebアプリです。

友達に推しを布教したいけど、何から説明すればいいかわからない。そんな時に、AIが推しの魅力を整理して、シェアしやすいWebページ形式で出力します。

## サービス開発の背景

Vtuberファンにとって「布教」は楽しみの一つです。しかし、推しの魅力を初見の人に伝えるのは意外と難しい。

- どの配信から見せればいい？
- 何が魅力なの？と聞かれてうまく言葉にできない
- 長文で説明しても読んでもらえない

OshiPitchは、これらの課題をAIの力で解決します。推しの名前を入れるだけで、ポイントを押さえた「布教資料」が完成。そのままリンクを送って共有できます。

## 機能紹介

### 🎤 プレゼン資料の自動生成

Vtuber名を入力すると、以下の情報を含むプレゼン資料を自動生成します。

- 基本プロフィール（名前・所属・デビュー日）
- キャッチコピー
- こんな人におすすめ（3つ）
- おすすめ動画（2〜3本）
- 名言・エピソード

### 🔗 シェア機能

生成した資料はWebページとして表示され、URLをそのまま友達に送れます。
